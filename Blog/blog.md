Here:

tag → current element

stat → loop metadata object

This stat object gives useful info about the loop.

Examples:

| Property     | Meaning               |
| ------------ | --------------------- |
| `stat.index` | index starting from 0 |
| `stat.count` | index starting from 1 |
| `stat.first` | true if first element |
| `stat.last`  | true if last element  |
