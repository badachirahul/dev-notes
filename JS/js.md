# JavaScript:

### 1️⃣ Node.js
* Node.js = JavaScript runtime
* It allows us to:
    - Run JS outside the browser
    - Build APIs
    - Create backend apps


### 2️⃣ nvm (Node Version Manager)
* nvm lets us switch between the versions easily.
* **Why needed?**
    - `Different projects may require different Node versions.`
* Example:
    ```js
    nvm install 18
    nvm use 18
    ```
* 🔁 Java Equivalent:
    - Like switching between Java 8, 11, 17 using SDK manager.


### 3️⃣ npm (Node Package Manager)
* Tool to install libraries.
* ex: `npm install express` - **It downloads from npm registry.** 
* 🔁 Java Equivalent: 
    - Like Maven/Gradle downloading from Maven Central.

### 4️⃣ package.json
* This is a configuration file.
* 🔁 Java Equivalent:
    - pom.xml
