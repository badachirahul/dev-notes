# Spring MVC Annotations 

## 1️⃣ Controller Annotations
- `@Controller`
- `@RestController`

    * **If we want to return a view page we use @Controller.**
    * **If we want to return JSON data directly to the client, then we use @RestController.** <br>
    `@RestController is commonly used when the frontend is built with frameworks like React or Angular.`

---

## 2️⃣ Request Mapping Annotations
- `@RequestMapping`
- `@GetMapping`
- `@PostMapping`
- `@PutMapping`
- `@DeleteMapping`
- `@PatchMapping`

    ### @RequestMapping - 
    * **@RequestMapping is a Spring MVC annotation used to map HTTP requests to controller classes or methods.**
    * **By default @RequestMapping allows all HTTP methods.**
        ```
        These annotations are just specialized versions of @RequestMapping:

        GET - Handles HTTP GET
        POST - Handles HTTP POST
        PUT - Handles HTTP PUT (Update)
        DELETE - Handles HTTP DELETE
        PATCH - Handles partial update
        ```
    * **Where @RequestMapping can be used?**
        - 1️⃣ **On class:**
            - `Used for base URL mapping.`

        - 2️⃣ **On method:**
            - `Used for specific endpoint mapping.`


    * **Example restricting method:**
        ```
        @RequestMapping(value="/hello", method=RequestMethod.GET)
        public String hello() {
            return "hello";
        }
        ```
        
        - **Why Spring created @GetMapping, @PostMapping**
            ```
            Because the above syntax was too verbose.

            So Spring introduced shortcuts.

            Example: @GetMapping("/hello")
           ```
    | Operation               | Annotation       | Meaning                 |
    | ----------------------- | ---------------- | ----------------------- |
    | Create                  | `@PostMapping`   | create new resource     |
    | Read                    | `@GetMapping`    | fetch data              |
    | Update (full)           | `@PutMapping`    | **Used for `update entire object`**    |
    | Update (partial / edit) | `@PatchMapping`  | **Used for `partial update`**          |
    | Delete                  | `@DeleteMapping` | delete resource         |

## 3️⃣ Request Data Binding Annotations
* **Used to extract data from request.**
    * `@RequestParam` - 
        - **Used to extract values from `query parameters` or `form data`**

        - **Query parameters -** `Query parameters are key-value pairs appended to the URL after ?`

    * `@PathVariable` - **Used to extract values from the URL path.**
    * `@RequestBody` - **Used to convert JSON request body → Java object.**
    * `@RequestHeader` - **Used to read HTTP headers.**
    * `@CookieValue` - **Used to read cookie values from request.**
