# Thymeleaf

### 4 main Thymeleaf expression types

| Expression | Reads From         | Example              |
| ---------- | ------------------ | -------------------- |
| `${}`      | Model              | `${name}`            |
| `@{}`      | URL building       | `@{/hello}`          |
| `*{}`      | Form object fields | `*{name}`            |
| `#{}`      | Messages file      | `#{welcome.message}` |
